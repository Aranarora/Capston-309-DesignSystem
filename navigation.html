Navigation

The navigation has a logo, navigation links, and a hamburger menu button. The links will be evaluated for space and will be hidden if they do not fit in the viewport, displaying the hamburger button instead. The navigation links will not appear on viewports smaller than 1024px. When the hamburger button is clicked, a navigation panel displaying the links will open.

Code behind Example:
<nav class="navbar"> <a href="home.html" style="width: 190px" ><img src="images/e-scooter.svg" style="width: 120px" alt="electric scooter image" /></a> <div id="myLinks"> <a href="#">Page 1</a> <a href="#">Page 2</a> <a href="#">Page 3</a> <a href="#">Page 4</a> </div> <a href="javascript:void(0);" class="icon" onclick="myFunction()"> <i class="fa fa-bars"></i> </a> </nav>

Html:

.navbar { background-color: #191b1a; position: relative; width : 100%; } .navbar #myLinks1 { display: none; } .navbar a { padding: 3%; color: whitesmoke; text-decoration: none; font-size: 1.6rem; display: block; } .navbar a.icon { background: #191b1a; display: block; position: absolute; right: 0; top: 0; padding: 4% 6%; margin: 16% 5%; } .navbar a:hover{background-color: #FE9C00; color: black;} .navbar a:active { background-color: #FE9C00; color: black; }

Css:

function myFunction() { var x = document.getElementById("myLinks"); if (x.style.display === "block") { x.style.display = "none"; } else { x.style.display = "block"; } }
