Navigation

The navigation has a logo, navigation links, and a hamburger menu button. The links will be evaluated for space and will be hidden if they do not fit in the viewport, displaying the hamburger button instead. The navigation links will not appear on viewports smaller than 1024px. When the hamburger button is clicked, a navigation panel displaying the links will open.

function myFunction() { var x = document.getElementById("myLinks"); if (x.style.display === "block") { x.style.display = "none"; } else { x.style.display = "block"; } }
